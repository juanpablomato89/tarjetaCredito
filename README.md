# TarjetaCredito — Credit Card UI Component

An Angular component that renders an interactive credit card UI with real-time form input reflection.

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | Angular, TypeScript |
| Styling | CSS3, Bootstrap |

## Features

- Animated card flip (front/back)
- Real-time card number, name, expiry, and CVV input reflection
- Card brand detection (Visa, MasterCard, etc.)
- Smooth CSS animations
- Responsive design
